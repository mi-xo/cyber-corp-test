## Overview

CyberShield Enterprise is a comprehensive, AI-powered cybersecurity training platform designed for government agencies, defense contractors, and enterprise organizations like BAE Systems. The platform provides a complete learning journey from Security Novice to Senior Cybersecurity Analyst, with real-world scenarios, hands-on labs, and industry-recognized certifications.

### Key Features

- **Complete Career Progression** - 7-tier advancement system from Novice to Principal
- **Comprehensive Training Modules** - 30+ modules covering all cybersecurity domains
- **Professional Certifications** - Industry-aligned certification programs
- **Red Team/Blue Team Exercises** - Realistic adversarial training scenarios
- **Hands-On Lab Environments** - Isolated environments for practical skills
- **Executive Analytics** - Compliance reporting and organizational insights
- **Threat Intelligence Integration** - Real-world threat feeds and MITRE ATT&CK
- **Team & Mentorship System** - Collaborative learning and knowledge transfer
- **Enterprise Security** - MFA, RBAC, audit logging, and compliance

## Technical Architecture

### Technology Stack

```
Frontend:
├── Next.js 14.2 (App Router)
├── React 18.3
├── TypeScript 5.4
├── Tailwind CSS 3.4
├── Framer Motion 11.0
└── Zustand 4.5

Backend:
├── Next.js API Routes
├── Prisma ORM 5.10
├── PostgreSQL 14+
├── Redis (caching)
└── NextAuth.js 4.24

AI/ML:
├── Anthropic Claude API
├── Scenario Generation
└── Real-time Coaching

Security:
├── MFA (TOTP)
├── RBAC
├── Audit Logging
├── Rate Limiting
└── Input Validation
```

### Security Features

- **Authentication**: NextAuth.js with MFA
- **Authorization**: 8-tier Role-Based Access Control
- **Clearance Levels**: 5-tier classification (Unclassified to TS/SCI)
- **Audit Logging**: Complete activity tracking
- **Rate Limiting**: Configurable per endpoint
- **Input Validation**: Zod schema validation
- **Encryption**: TLS 1.3, AES-256 at rest

## Installation

### Prerequisites

- Node.js 18.0+
- PostgreSQL 14+
- Redis 7+ (optional)
- Docker (for labs)

### Quick Start

```bash
# Clone the repository
git clone https://github.com/your-org/cybershield-enterprise.git
cd cybershield-enterprise

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local
# Edit .env.local with your settings

# Database setup
npm run db:generate
npm run db:migrate
npm run db:seed

# Start development server
npm run dev
```

## API Documentation

### Core Endpoints

```
Authentication:
POST /api/auth/signin          Sign in
POST /api/auth/register        Register
POST /api/auth/mfa/setup       Setup MFA
POST /api/auth/mfa/verify      Verify MFA

Training:
GET  /api/training/modules     List modules
GET  /api/training/modules/:id Get module
POST /api/training/sessions    Start session
POST /api/scenarios/generate   Generate AI scenario

Progress:
GET  /api/progress             User progress
GET  /api/progress/analytics   Analytics data
GET  /api/leaderboard          Rankings

Certifications:
GET  /api/certifications       List certifications
POST /api/certifications/exam  Start exam
GET  /api/certifications/verify Verify cert

Admin:
GET  /api/admin/users          List users
GET  /api/admin/analytics      Org analytics
GET  /api/admin/compliance     Compliance reports
```
